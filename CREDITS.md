### Libraries & Tools
This project wouldn't be possible without these amazing open-source libraries:
* **[Raylib](https://www.raylib.com/)** - An amazing, simple and easy-to-use library to enjoy videogames programming. Raylib made the UI and rendering incredibly straightforward and fun to work with. Huge thanks to [Ramon Santamaria (@raysan5)](https://github.com/raysan5) and all contributors!
* **[libmpv](https://github.com/mpv-player/mpv)** - Powerful video playback engine that powers the video functionality
* **[FFmpeg](https://ffmpeg.org/)** - Essential multimedia framework for thumbnail generation and video processing
* **[cJSON](https://github.com/DaveGamble/cJSON)** - Lightweight JSON parser for settings persistence
* **[Zenity](https://gitlab.gnome.org/GNOME/zenity)** - Simple dialog tool for the file picker interface
